# Réactions d'oxydoréduction

## Objectifs

À la fin de cette leçon, tu sauras :
- Identifier oxydants et réducteurs
- Équilibrer les demi-équations redox
- Prévoir le sens des réactions redox
- Décrire le fonctionnement d'une pile

## Définitions

:::definition
**Oxydant et réducteur**

- Un **oxydant** est une espèce capable de **gagner** des électrons
- Un **réducteur** est une espèce capable de **perdre** des électrons

Demi-équation d'un couple Ox/Red :
$$\text{Ox} + n\text{e}^- \rightleftharpoons \text{Red}$$
:::

:::definition
**Réaction d'oxydoréduction**

Une réaction redox est un transfert d'électrons du réducteur d'un couple vers l'oxydant d'un autre couple.

- **Oxydation** : perte d'électrons (Red → Ox + ne⁻)
- **Réduction** : gain d'électrons (Ox + ne⁻ → Red)
:::

## Couples redox courants

:::property[Couples importants]
| Couple Ox/Red | Demi-équation |
|---------------|---------------|
| MnO₄⁻/Mn²⁺ | MnO₄⁻ + 8H⁺ + 5e⁻ = Mn²⁺ + 4H₂O |
| Cr₂O₇²⁻/Cr³⁺ | Cr₂O₇²⁻ + 14H⁺ + 6e⁻ = 2Cr³⁺ + 7H₂O |
| Fe³⁺/Fe²⁺ | Fe³⁺ + e⁻ = Fe²⁺ |
| I₂/I⁻ | I₂ + 2e⁻ = 2I⁻ |
| Cu²⁺/Cu | Cu²⁺ + 2e⁻ = Cu |
| Zn²⁺/Zn | Zn²⁺ + 2e⁻ = Zn |
| H⁺/H₂ | 2H⁺ + 2e⁻ = H₂ |
:::

## Équilibrage des demi-équations

:::method[Méthode d'équilibrage]
1. Équilibrer l'élément principal (autre que O et H)
2. Équilibrer l'oxygène avec H₂O
3. Équilibrer l'hydrogène avec H⁺
4. Équilibrer les charges avec e⁻
:::

## Équation bilan

:::method[Écrire l'équation bilan]
1. Écrire les deux demi-équations
2. Multiplier pour avoir le même nombre d'électrons
3. Additionner les demi-équations
4. Les électrons s'éliminent
:::

## Pile électrochimique

:::definition
**Pile**

Une **pile** est un générateur qui convertit l'énergie chimique en énergie électrique via une réaction redox spontanée.

Composants :
- **Anode** (-) : électrode où a lieu l'oxydation
- **Cathode** (+) : électrode où a lieu la réduction
- **Pont salin** : assure la continuité électrique
:::

:::property[Fonctionnement]
- À l'anode : Red₁ → Ox₁ + ne⁻ (oxydation)
- À la cathode : Ox₂ + ne⁻ → Red₂ (réduction)
- Les électrons circulent de l'anode vers la cathode dans le circuit extérieur
:::

:::property[Force électromotrice (f.e.m.)]
La f.e.m. E d'une pile dépend des couples mis en jeu :
$$E = E_{cathode} - E_{anode}$$

Plus le pouvoir oxydant de Ox₂ est grand et le pouvoir réducteur de Red₁ est grand, plus E est élevée.
:::

## Électrolyse

:::definition
**Électrolyse**

L'**électrolyse** est une transformation forcée (non spontanée) par apport d'énergie électrique.

- Anode (+) : oxydation forcée
- Cathode (-) : réduction forcée

C'est l'inverse d'une pile.
:::

## Exemples résolus

### Exemple 1 : Équilibrage

Équilibrer : MnO₄⁻ → Mn²⁺ (en milieu acide)

1. Mn est déjà équilibré
2. Oxygène : MnO₄⁻ → Mn²⁺ + 4H₂O
3. Hydrogène : MnO₄⁻ + 8H⁺ → Mn²⁺ + 4H₂O
4. Charges : gauche = -1 + 8 = +7, droite = +2

   Ajout de 5e⁻ à gauche : **MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O**

### Exemple 2 : Équation bilan

Réaction entre MnO₄⁻ et Fe²⁺.

Couples : MnO₄⁻/Mn²⁺ et Fe³⁺/Fe²⁺

Demi-équations :
- MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O (×2)
- Fe²⁺ → Fe³⁺ + e⁻ (×10)

Bilan : **2MnO₄⁻ + 16H⁺ + 10Fe²⁺ → 2Mn²⁺ + 8H₂O + 10Fe³⁺**

### Exemple 3 : Pile Daniell

Pile Cu/Zn : couples Cu²⁺/Cu et Zn²⁺/Zn

- Le zinc est plus réducteur que le cuivre
- Anode (Zn) : Zn → Zn²⁺ + 2e⁻
- Cathode (Cu) : Cu²⁺ + 2e⁻ → Cu

Bilan : **Zn + Cu²⁺ → Zn²⁺ + Cu**

## Erreurs fréquentes

:::warning
**Erreur 1** : Confondre oxydation et réduction

Oxydation = perte d'e⁻ (le n.o. augmente)
Réduction = gain d'e⁻ (le n.o. diminue)

**Erreur 2** : Oublier d'équilibrer les charges

Toujours vérifier que les charges sont égales de chaque côté.

**Erreur 3** : Inverser anode et cathode

Pile : anode (-), cathode (+)
Électrolyse : anode (+), cathode (-)

**Erreur 4** : Oublier le pont salin

Sans pont salin, pas de circuit fermé, pas de pile !
:::

## Mini-résumé

- **Oxydant** : gagne e⁻ | **Réducteur** : perd e⁻
- Équilibrage : élément → O (H₂O) → H (H⁺) → charges (e⁻)
- **Pile** : réaction spontanée, anode (-) oxydation, cathode (+) réduction
- **Électrolyse** : réaction forcée, c'est l'inverse
