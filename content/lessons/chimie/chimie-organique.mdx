# Chimie organique : groupes caractéristiques

## Objectifs

À la fin de cette leçon, tu sauras :
- Identifier les principaux groupes caractéristiques
- Nommer les molécules organiques
- Reconnaître les familles de composés

## Intuition et analogie

Imagine des **LEGO** : tu as des briques de base (le carbone et l'hydrogène) et des pièces spéciales colorées (les groupes fonctionnels). La "personnalité" de ta construction dépend surtout de ces pièces spéciales !

En chimie organique, le **squelette carboné** (la chaîne de carbones) est comme la structure de base en briques grises. Ce qui rend chaque molécule unique, c'est le **groupe caractéristique** qu'elle porte :
- Un **–OH** (alcool) → la molécule peut former des liaisons hydrogène, elle est soluble dans l'eau (comme l'éthanol)
- Un **–COOH** (acide) → la molécule est acide, elle peut réagir avec des bases (comme le vinaigre)
- Un **–NH₂** (amine) → la molécule est basique, elle a souvent une odeur caractéristique

La **nomenclature**, c'est le "langage" pour décrire ces constructions : tu comptes les briques du squelette (méth = 1, éth = 2, prop = 3...) et tu ajoutes le suffixe du groupe (-ol, -al, -one...).

Ainsi, **éthanol** = 2 carbones (éth-) + alcool (-ol). **Propanone** (acétone) = 3 carbones (prop-) + cétone (-one). Logique comme un code LEGO !

## Groupes caractéristiques

:::definition
**Groupe caractéristique**

Un **groupe caractéristique** (ou groupe fonctionnel) est un groupe d'atomes qui confère des propriétés chimiques spécifiques à une molécule.
:::

:::property[Principaux groupes]
| Groupe | Famille | Formule | Suffixe/Préfixe |
|--------|---------|---------|-----------------|
| –OH | Alcool | R–OH | -ol |
| –CHO | Aldéhyde | R–CHO | -al |
| –CO– | Cétone | R–CO–R' | -one |
| –COOH | Acide carboxylique | R–COOH | acide ...-oïque |
| –COO– | Ester | R–COO–R' | -oate de ...-yle |
| –NH₂ | Amine | R–NH₂ | -amine |
| –CONH₂ | Amide | R–CONH₂ | -amide |
:::

## Nomenclature

:::method[Règles de nomenclature]
1. Trouver la **chaîne carbonée principale** (la plus longue contenant le groupe prioritaire)
2. Numéroter pour que le groupe prioritaire ait le plus petit indice
3. Nommer : préfixes + chaîne + suffixe du groupe

**Préfixes** : méth- (1C), éth- (2C), prop- (3C), but- (4C), pent- (5C), hex- (6C)...
:::

:::property[Ordre de priorité des groupes]
Du plus prioritaire au moins prioritaire :
1. Acide carboxylique (-oïque)
2. Ester (-oate)
3. Amide (-amide)
4. Aldéhyde (-al)
5. Cétone (-one)
6. Alcool (-ol)
7. Amine (-amine)
:::

## Les alcools

:::definition
**Alcool**

Un alcool possède le groupe hydroxyle –OH lié à un carbone tétragonal.

Classes :
- **Primaire** : –CH₂OH
- **Secondaire** : –CHOH–
- **Tertiaire** : C–OH avec 3 chaînes carbonées
:::

## Les composés carbonylés

:::definition
**Aldéhyde et cétone**

- **Aldéhyde** : groupe C=O en bout de chaîne (–CHO)
- **Cétone** : groupe C=O au milieu de la chaîne (–CO–)

Le carbone du C=O est trigonal plan.
:::

## Les acides et esters

:::definition
**Acide carboxylique**

Le groupe carboxyle –COOH peut libérer un H⁺ (acidité).

Exemple : acide éthanoïque (acide acétique) CH₃COOH
:::

:::definition
**Ester**

Un ester résulte de la réaction entre un acide et un alcool :
$$\text{R–COOH} + \text{R'–OH} \rightleftharpoons \text{R–COO–R'} + \text{H}_2\text{O}$$

Nomenclature : (nom de l'acide en -oate) de (nom de l'alcool en -yle)

Exemple : éthanoate de méthyle CH₃COOCH₃
:::

## Les amines et amides

:::definition
**Amine**

Une amine possède un atome d'azote lié à 1, 2 ou 3 groupes carbonés.

- Amine primaire : R–NH₂
- Amine secondaire : R–NH–R'
- Amine tertiaire : R–N(R')–R''
:::

:::definition
**Amide**

Un amide possède le groupe –CO–NH₂ (ou –CO–NHR ou –CO–NR₂).

Liaison peptidique : –CO–NH– (amide secondaire)
:::

## Exemples résolus

### Exemple 1 : Nomenclature

CH₃–CH₂–CH₂–OH

- Chaîne : 3 carbones → prop-
- Groupe : –OH alcool → -ol
- Position : carbone 1

**Nom : propan-1-ol**

### Exemple 2 : Cétone

CH₃–CO–CH₂–CH₃

- Chaîne : 4 carbones → but-
- Groupe : C=O cétone → -one
- Position : carbone 2

**Nom : butan-2-one**

### Exemple 3 : Ester

CH₃–COO–CH₂–CH₃

- Partie acide : CH₃–CO– (acide éthanoïque) → éthanoate
- Partie alcool : –O–CH₂–CH₃ (éthanol) → éthyle

**Nom : éthanoate d'éthyle**

### Exemple 4 : Identification

Formule C₃H₆O₂ avec un groupe –COO–.

C'est un **ester** : méthanoate d'éthyle (HCOO–CH₂–CH₃)

## Erreurs fréquentes

:::warning
**Erreur 1** : Confondre aldéhyde et cétone

Aldéhyde : –CHO en bout de chaîne
Cétone : –CO– au milieu

**Erreur 2** : Mal numéroter la chaîne

Toujours numéroter pour minimiser l'indice du groupe prioritaire.

**Erreur 3** : Inverser les parties de l'ester

Ester : R–COO–R' → (acide)-oate de (alcool)-yle

**Erreur 4** : Confondre amine et amide

Amine : –NH₂ seul | Amide : –CO–NH₂ (azote lié à un carbonyle)
:::

## Mini-résumé

- **Alcool** : –OH → -ol
- **Aldéhyde** : –CHO → -al
- **Cétone** : –CO– → -one
- **Acide** : –COOH → acide ...-oïque
- **Ester** : –COO– → -oate de ...-yle
- **Amine** : –NH₂ → -amine
- **Amide** : –CONH₂ → -amide
